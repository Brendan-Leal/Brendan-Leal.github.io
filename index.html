More about Sentinel

With Sentinel, deployments are easy, and updates are safe. Sentinel core features are accessible through a convenient command line interface to assist developers in upgrading their applications with confidence via canary deployments. Sentinel's cloud-agnostic and open-source design ensures developers retain control and flexibility over their applications.
